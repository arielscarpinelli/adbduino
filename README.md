Yet another ADBduino
====================


Tunned to work with an Arduino Nano and some old Genius PS2 mouse.

Pinout
======

Arduino D3 -> ADB pin 1 (Data)
Arduino D6 -> PS2 Mouse pin 1 (Data)
Arduino D7 -> PS2Mouse pin 5 (Clock)

GND -> ADB pin 4 
GND -> PS2 Mouse pin 3

+5v -> ADB pin 3
+5v -> PS2 Mouse pin 4
